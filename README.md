# Landing Page Espaço Som

Landing page estática para captação de contatos via WhatsApp e agendamento de aula experimental de guitarra e violão.

## Estrutura

- `index.html`: marcação semântica da página.
- `css/styles.css`: estilos mobile first.
- `js/script.js`: interações, consentimento de cookies e eventos preparados.
- `config.js`: configurações usadas pela página.
- `site.config.json`: registro das configurações do projeto.
- `assets/`: pasta reservada para logo, favicon, fotos e vídeos futuros.

## Como visualizar

Abra `index.html` no navegador.

## Rastreamento

A estrutura está preparada para:

- Google Tag Manager;
- Google Analytics 4;
- Meta Pixel;
- Google Ads;
- eventos de CTA, scroll, visualização de oferta e metodologia.

Os IDs estão vazios em `config.js` e `site.config.json`. Não foram usados IDs fictícios.
