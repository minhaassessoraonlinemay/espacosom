# Objetivo

Este documento centraliza o estado atual do projeto.

Ele deve ser o primeiro arquivo consultado ao abrir um projeto, permitindo que qualquer pessoa ou inteligência artificial compreenda rapidamente:

- em que fase o projeto está;
- o que já foi concluído;
- o que falta fazer;
- quais são os bloqueios;
- qual é o próximo passo.

Sempre mantenha este documento atualizado.
