# Objetivo

Este documento centraliza toda a estratégia de comunicação do projeto.

Seu objetivo é organizar a copy de forma estruturada para que qualquer pessoa ou inteligência artificial consiga compreender como a comunicação deve ser construída antes de iniciar o desenvolvimento da página.

Toda decisão relacionada à comunicação deve partir deste documento.
